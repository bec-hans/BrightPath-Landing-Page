# BrightPath Learning - E-Learning Landing Page

A premium, high-impact landing page for an e-learning business, designed with a sleek, dark-themed aesthetic inspired by Spotify's web interface.

## 🚀 Features

- **Spotify-Inspired UI:** Dark mode aesthetic with high-contrast accents and professional typography.

- **Interactive Category Grid:** "Spotify-style" cards that expand to show details and specific course calls-to-action without leaving the page.

- **Guided Career Paths:** High-resolution visual thumbnails for specialized tracks like UX Design, Fullstack Development, and Data Analytics.

- **Responsive Design:** Fully fluid layout optimized for mobile, tablet, and desktop viewing.

- **Modern Tech Stack:** Built with HTML5, Tailwind CSS for styling, and Vanilla JavaScript for interactive components.

<br>

## 🛠️ Technology Used

- HTML5

- Tailwind CSS (via CDN)

- Google Fonts (Inter)

- Vanilla JavaScript

<br>

## 📂 Project Structure

- index.html: The core landing page containing all HTML structure, Tailwind styling, and interaction logic.

- README.md: Project documentation.
